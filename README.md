# User Management API

## Main Features

* User Signup
* User Login
* Get Users
* Delete User

## Tech Stack

* Node.js
* Express
* MongoDB
* Mongoose

## Endpoints

### Signup

POST /users/signup

### Login

POST /users/login

### Get Users

GET /users

### Delete User

DELETE /users/:id

## How to Run

1. npm install
2. node index.js
